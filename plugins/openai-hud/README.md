# OpenAI HUD

Repo-local plugin for quick context and status inspection.

## Included

- `.codex-plugin/plugin.json`
- `skills/project-hud/SKILL.md`
- `scripts/collect-status.mjs`
- `scripts/collect-workspace-status.mjs`

## Example

```bash
node plugins/openai-hud/scripts/collect-status.mjs
node plugins/openai-hud/scripts/collect-workspace-status.mjs
```
