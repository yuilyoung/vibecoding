# Skill: Review Against Baseline

## 목적

구현 결과가 현재 활성 베이스라인과 handoff 기준을 만족하는지 `울트론`이 검토할 때 사용하는 기준이다.

## 확인 항목

- 활성 실행 베이스라인: `work/2D-FPS-game`
- handoff 범위 초과 변경 여부
- 문서 기준과 코드 기준 충돌 여부
- 테스트/빌드 결과
- 다음 handoff로 넘길 리스크 유무

## 출력 형식

- `done`: 충족된 항목
- `risk`: 남은 리스크
- `followup`: 비전이 다음 계획에서 다뤄야 할 것
