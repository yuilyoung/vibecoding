# 문서 작성 강제 규칙

## doc-writer 독점 (강제)

- **모든 문서 생성/수정은 doc-writer 에이전트만 수행한다.**
- 다른 에이전트가 문서가 필요한 경우 doc-writer에게 메일박스로 요청한다.
- PM 보고서(docs/reports/)도 doc-writer가 작성하되, PM이 내용을 지시한다.

## docs/ 폴더 관리 (강제)

- 모든 MD 문서는 `docs/` 하위 분류 폴더에 저장한다.
- 루트 또는 임의 경로에 문서 생성 금지.
- 새로운 주제는 적절한 하위 폴더를 생성하여 분류한다.

## 200줄 제한 (강제)

- 모든 문서는 **200줄 이하**로 작성한다.
- 200줄 초과 시 핵심 문서 + 상세 참조 문서로 분리한다.
- 참조 링크 형식: `[상세 내용](./path/to/detail.md)`
- 코드 예제, 표, 다이어그램도 줄 수에 포함된다.

## 참조 링크 규칙

| 대상 | 참조 방식 |
|------|-----------|
| 긴 코드 예제 | 별도 파일 분리 후 링크 |
| 관련 문서 | 상대 경로로 교차 참조 |
| 외부 리소스 | URL 링크 |
| 상세 스펙 | 별도 상세 문서로 분리 후 링크 |
